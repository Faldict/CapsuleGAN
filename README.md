# CapsuleGAN

## Quick Start
To run this code, you should open your terminal and type these commands.

```
git clone https://github.com/Faldict/CapsuleGAN
cd CapsuleGAN
python main.py
```

## Requirements

- python3
- numpy
- tensorflow (gpu better)
